DynamicBone

1. DynamicBone参数含义

  • Damping
    阻尼系数: 代表骨骼受到的阻力。

    How much the bones slowed down.
    描述振幅随时间减小的快慢，用于限制跟随速度的大小。阻尼系数越小衰变越慢，阻尼振动时间越长。

  • Elasticity
    弹性，产生形变后恢复原状的能力。

    How much the force applied to return each bone to original orientation.

  • Siffness
    刚度，骨骼保持原位的能力。

    How much bone’s original orientation are preserved.
    这个值越趋近于1，产生单位位移所需的力则越大，表现得越趋近于刚体。当这个值越趋近于0，表现得越像绳子。

  • Inert
    角色有多少位移不用于物理模拟

  • Radius
    每根骨骼球形碰撞体的大小，Radius 代表球体的半径。
  • Damping Distrib / Elasticity Distrib / Siffness Distrib
    参数变化曲线，曲线的值乘以参数值进行插值的。
  • End Length
    如果这个值不为0，说明有一根额外的骨骼在hierarchy面板中了。length将会乘以两根骨骼的距离
  • End Offset
    角色localPosition的偏移量
  • Gravity
    世界坐标系的重力加速度
  • Force
    施加在骨骼上的力
  • Colliders
    骨骼的碰撞体，骨骼不会穿透碰撞体。
  • Exclusions
    骨骼禁用物理模拟
  • Freeze Axis
    试骨骼在指定平面运动
  • Distant Disable / Reference Object / Distance To Object
    自动禁用物理模拟，当角色离相机或者玩家太远。如果没有参照物，则使用主相机作为参照物。

    2. DynamicBoneCollider参数含义

  • Center
    球体或者胶囊体的重心，使用的角色的自身坐标系。
  • Height
    胶囊体的高度，包含了两个半球的高度。且会随着角色的缩放而缩放。
  • Direction
    胶囊体的高度轴线。
  • Bound
    强制使用骨骼的外边界或者内边界。
